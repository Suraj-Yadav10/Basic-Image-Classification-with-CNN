# Basic-Image-Classification-with-CNN
Image classification with CNN is when a computer learns to identify objects in images. A CNN looks at small features like edges and colors, then combines them to recognize shapes and objects, finally predicting a label like cat or car.

A CNN (Convolutional Neural Network) is a type of deep learning model especially good at extracting patterns from images, like edges, textures, shapes, etc.

# Functionality of CNN 

A typical CNN pipeline:

Convolution Layers
Apply filters to detect features (edges, patterns)
Activation (ReLU)
Adds non-linearity
Pooling (MaxPooling)
Reduces size, keeps important info
Fully Connected Layer
Makes final classification

Think of it like:

Pixels → Edges → Shapes → Objects → Label
